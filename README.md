# Autopilot Test Repo

Test repository for validating the autopilot skill.

## Project Overview

This project serves as an end-to-end test environment for the autopilot skill. It provides a controlled repository where the autopilot orchestration flow can be validated, from PRD ingestion through issue creation, implementation, and verification.

## Installation

```bash
git clone git@github.com:cheong38/autopilot-test-repo.git
cd autopilot-test-repo
```

## Usage

```bash
# Run the autopilot skill against the test PRD
/autopilot test-prd.md
```
